2004年世界艾滋病运动：妇女、女童与艾滋病 (上)

 

 

2004年世界艾滋病运动主题是妇女、女童与艾滋病。联合国艾滋病规划署就此发表策略概述，以下是部分摘录。

 

 

为什么把妇女作为重点？

 

妇女和女童特别容易受艾滋病感染和艾滋病的影响。艾滋病感染者中约一半为女性。

 

在全世界，年轻人中的艾滋病感染率正在迅速上升―在发展中国家，67%的新感染者是15－24岁的年轻人。感染艾滋病的风险越来越大的现象，在15~24岁的年轻妇女和女童中特别明显，在发展中国家的艾滋病感染者和患者中，她们所占的比例高达64%。

 

在全球范围内，年轻妇女和女童与男人和男童相比更容易感染艾滋病，研究表明她们感染艾滋病的机会是同龄男性的2.5倍。在撒哈拉以南的非洲，女童和妇女感染艾滋病的机会是同龄男性的两倍，在某些地区，女童和妇女的艾滋病感染率为同龄男性的6倍。在东部非洲和南部非洲的部分地区，1/3以上的十几岁的女童感染了艾滋病。在某些加勒比国家，也在出现这样的趋势。

 

自20年前艾滋病开始流行以来，至今已有3000万人死于艾滋病，现有感染者4000多万人。在贫困国家，600万艾滋病感染者和患者需要接受抗逆转录病毒治疗，在需要得到照顾、治疗和支持的人中，妇女和儿童占了很大比例。

 

与男子相比，妇女从单次无保护性交中感染艾滋病的机会是男性的两倍，然而她们仍然依赖男性的合作，保护她们免受感染。此外，在世界范围内，人们仍期望妇女在从事家务劳动和照顾家庭成员方面起带头作用。对许多妇女来说，艾滋病已显著增加了她们照顾家人的负担。贫困和恶劣的公共服务也已经与艾滋病结合起来，将压在妇女肩上的照顾艾滋病感染者和患者的负担转变成了一场危机，这场危机对社会、卫生和经济方面产生的后果是深重的。

 

艾滋病加剧了贫困的女性化，在受到艾滋病沉重打击的国家更是如此。艾滋病还使得妇女丧失了她们的权利。妇女为照顾患艾滋病的家庭成员，因而在家庭里从事创收劳动的时间减少了，全家人因脆弱性的增加，因此都受到了严重的影响。

 

过去10年内，在艾滋病高流行国家，女童入学率已大为下降。女童首先被迫离开学校以护理患病的亲人，或照顾年幼的弟妹。艾滋病正威胁着近年来在基础教育方面取得的成果，并且不成比例地影响着女童的小学入学率。

 

上学读书就是一种保护。教育是抵御艾滋病传播及其影响的关键手段之一，这方面的证据越来越多。

 

世界艾滋病运动努力促使人们更加认识到艾滋病对妇女和女童的影响，并帮助人们解决这些问题。

 

 

这个主题是如何决定的？

 

世界艾滋病运动必须确保其2004年的主题与《承诺宣言》确定的重点相吻合，因为2001通过的该宣言为今后几年的世界艾滋病运动提供了框架。

 

在2003年夏季，联合国艾滋病规划署咨询了2500多个非政府组织、联合国艾滋病规划署驻各国办事处代表、以及其共同发起组织的工作人员，并通报了2004年世界艾滋病主题的确定过程。约100个团体和机构作出了应答。一些团体和机构建议将重点放在注射吸毒者，另一些则建议将家庭作为重点。而绝大部分应答者要求将重点放在妇女与艾滋病上。

 

除了这些咨询活动外，刚刚成立的由民间团体代表组成的世界艾滋病运动临时指导委员会批准了该主题。 委员会的任务是帮助设计和加强世界艾滋病运动。 委员会此后成为了世界艾滋病运动全球指导委员会。

 

把妇女作为重点的决定受到了联合国艾滋病规划署和其他联合国机构的欢迎，因为该重点与联合国大会艾滋病特别会议《承诺宣言》的关键目标紧密相连。如果这些目标达到了，妇女和女童保护自己免受艾滋病感染和应对其后果的能力都将大为提高。

 

对咨询作出反馈的应答者提出了需要立即优先处理的重点问题：

 

妇女和女童在艾滋病感染者中所占比例越来越大，在某些地区，尤其是撒哈拉以南的非洲地区，妇女占感染者的一半以上。同时，艾滋病流行造成的社会和经济影响对妇女和女童产生的影响特别大，照顾艾滋病感染者和患者的大部分工作已落在妇女和女童身上。对咨询作出应答者还指出：

 

n             妇女在发展中发挥着核心作用

n             性别在驱动艾滋病流行中起关键作用

n             与男性相比，妇女更容感染艾滋病病毒

n             在艾滋病感染最严重地区，50%以上的感染者是妇女和女童

n             侮辱和歧视－基于现存的不平等，仍在加剧艾滋病的流行

n             妇女能够鼓舞人们，向所有人提供力量和希望

 

艾滋病流行会给妇女和女童带来毁灭性的影响并不是什么新现象，也不是新认识到的现象。

 

新成立的全球妇女与艾滋病联盟，这是联合国艾滋病规划署于2004年1月发起的，其目的是将各个地方的活动与全球性的活动结合起来，找出差距，进一步加强正在进行的工作。联合国艾滋病规划署将全力支持增强妇女和女童抵御感染艾滋病的能力，以及改善艾滋病感染者和病人生活质量的相关行动和宣传活动。

 

艾滋病流行给妇女和女童将带来影响，因此，防治工作其他方面取得的长期进展取决于今天我们对妇女和女童所做的工作。
